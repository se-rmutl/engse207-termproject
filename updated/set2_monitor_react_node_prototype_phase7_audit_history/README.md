# ENGSE207 Set 2 Monitor — Phase 7 Audit Log

This package extends the Set 2 monitor with audit log / history features.

## New in Phase 7
- audit_logs table in SQLite
- teacher can view group history
- student can view own group history
- events logged for login, setup, submission save, status change, note update, teacher review, and health checks

## Run
```bash
npm install
npm start
```

Open http://localhost:8080

Teacher: `teacher / teacher123`
Group example: `sec1-group01 / group01pass`
